How To Play Lyricle

Objective

Guess the hidden lyrics to SONG by ARTIST!

Instructions
  • The first line of the secret lyrics are revealed to get you started.
  • Click into a lyric input field and start typing your guesses!
  • The size of each input field is relative to how long the lyric is.
  • Your input will start to change color as you get closer to the secret lyric, so keep guessing and don't give up!
  • If you get stuck, use a lifeline! Select the word you want to reveal, then press and hold ♥ on the on-screen keyboard. Once you use your 3rd and final lifeline, the game is over. Choose wisely!
  • Don't worry about punctuation or capitalization, just focus on the lyrics!
  • The game is over when you've entered all of the correct lyrics or used all 3 of your lifelines.
Scoring

Scoring is based on number of guessed letters, time to completion, and remaining lifelines.


Welcome to Lyricle! The game where you guess the lyrics to a song. You can use a lifeline to reveal a word in the lyrics. If you guess all the lyrics correctly or use up all of your lifelines, the game is over!

To use a lifeline, select the word you want to be given, then click "Use a Lifeline". Guess all the lyrics correctly or use up all of your lifelines and the game is over!
