Lyricle

Welcome to Lyricle! The game where you guess the lyrics to a song. You can use a lifeline to reveal a word in the lyrics. If you guess all the lyrics correctly or use up all of your lifelines, the game is over!

To use a lifeline, select the word you want to be given, then click "Use a Lifeline". Guess all the lyrics correctly or use up all of your lifelines and the game is over!
